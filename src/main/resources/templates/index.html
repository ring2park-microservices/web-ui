Ring2Park Microservices Demo

Overview

  • Aa simple web application for accessing parking locations and customer user data.
  • All the parking location and user information is fetched via a RESTful interface to relevant microservices.

Demo

  • Eureka Dashboard: http://@eureka-server@:@eureka-port@
  • Check applications registered: http://@eureka-server@:@eureka-port@/eureka/apps/
  • Fetch all users: /users
  • Fetch a user by specific username: /users/username/fred
  • User Search

Spring Boot URLs

Spring Boot RESTful URLs for interrogating the application (they return JSON format data):

  • The environment
